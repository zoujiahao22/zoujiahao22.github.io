华为机试题输入输出总结

华为机试题采用的是ACM模式，需要考生自行编写输入和输出，对于已经习惯了只编写函数体部分的考生来说可能会是个挑战，本人尝试根据自己在刷华为机试题过程中遇到的各种输入输出的case进行梳理，希望能够帮助大家尽快适应华为机试题的输入输出模式，减少在这块的耗时，从而将更多的精力放在解题上。

一 基础篇

首先对与Java的输入，要用Scanner，因此代码起手第一句，一定得先来上这么一段：

Scanner scaner = new Scanner(System.in);

为了方便我们不重新重新启动程序的情况下，可以反复地测试多组测试应用，我们通常会写一个while循环来反复接受输入，那么while循环的条件判断应该怎么写呢？

读取下一个整数/字符/行：

  1. // 判断是否有下一个整数
  2. while(scanner.hasNextInt()){
  3.   //...
  4.    scanner.nextInt();
  5. }
  6.  
  7. // 判断是否有下一个字符
  8. while(scanner.hasNext()){
  9.    //...
  10.    scanner.next();
  11. }
  12.  
  13. // 判断是否有一整行
  14. scanner.hasNextLine(){
  15.    //...
  16.    scanner.nextLine();
  17. }

如果采用has xxxx的话，后面也要用next xxxx。比如前面用hasNextLine，那么后面要用 nextLine 来处理输入。

二 正文篇

1. 多组空格分割的两个整数 （无行数，结束字符限制）

题目描述：

输入包括两个正整数a,b(1 <= a, b <= 10^9),输入数据包括多组。
输出a+b的结果

输入示例：

  1. 1 5
  2. 10 20

输出示例

  1. 6
  2. 30
  1. import java.util.Scanner;
  2. public class Main {
  3.     public static void main(String[] args) {
  4.         Scanner in = new Scanner(System.in);
  5.         while (in.hasNext()) { 
  6.             int a = in.nextInt();
  7.             int b = in.nextInt();
  8.             System.out.println(a + b);
  9.         }
  10.     }
  11. }

2. 第一行组数接空格分割的两个正整数 （行数限制）

题目描述：

输入第一行包括一个数据组数t(1 <= t <= 100)
接下来每行包括两个正整数a,b(1 <= a, b <= 10^9)
输出a+b的结果

输入示例：

  1. 2
  2. 1 5
  3. 10 20

输出示例：

  1. 6
  2. 30
  1. import java.util.Scanner;
  2. public class Main {
  3.     public static void main(String[] args) {
  4.         Scanner in = new Scanner(System.in);
  5.         int num = in.nextInt();
  6.         for(int i = 0; i < num; i++) { // 注意 while 处理多个 case
  7.             int a = in.nextInt();
  8.             int b = in.nextInt();
  9.             System.out.println(a + b);
  10.         }
  11.     }
  12. }

3. 空行分割的两个正整数，0 0 结束 （结束符限制）

题目描述：

输入包括两个正整数a,b(1 <= a, b <= 10^9),输入数据有多组, 如果输入为0 0则结束输入
输出a+b的结果

输入示例：

  1. 1 5
  2. 10 20
  3. 0 0

输出示例：

  1. 6
  2. 30
  1. import java.util.Scanner;
  2. public class Main {
  3.     public static void main(String[] args) {
  4.         Scanner in = new Scanner(System.in);
  5.         
  6.         while (in.hasNext()) { 
  7.             int a = in.nextInt();
  8.             int b = in.nextInt();
  9.             if(a ==0 && b == 0) break;
  10.             System.out.println(a + b);
  11.         }
  12.     }
  13. }

4. 每行第一个为个数后带空格分割整数，0结束 （结束符限制,每行有个数限制）

题目描述：

输入数据包括多组。
每组数据一行,每行的第一个整数为整数的个数n(1 <= n <= 100), n为0的时候结束输入。
接下来n个正整数,即需要求和的每个正整数。

每组数据输出求和的结果

输入示例：

  1. 4 1 2 3 4
  2. 5 1 2 3 4 5
  3. 0

输出示例：

  1. 10
  2. 15
  1. import java.util.Scanner;
  2. public class Main {
  3.     public static void main(String[] args) {
  4.         Scanner in = new Scanner(System.in);
  5.         
  6.         while (in.hasNext()) { 
  7.             int n = in.nextInt();
  8.             if(n == 0) break;
  9.             int sum = 0;
  10.             for (int i = 0; i < n; i++) {
  11.                 sum += in.nextInt();
  12.             }
  13.             System.out.println(sum);
  14.         }
  15.     }
  16. }

5. 第一行组数接第一个个数接空格分开的整数 （行数限制,每行有个数限制）

题目描述：

输入的第一行包括一个正整数t(1 <= t <= 100), 表示数据组数。
接下来t行, 每行一组数据。
每行的第一个整数为整数的个数n(1 <= n <= 100)。
接下来n个正整数, 即需要求和的每个正整数。

每组数据输出求和的结果

输入示例：

  1. 2
  2. 4 1 2 3 4
  3. 5 1 2 3 4 5

输出示例：

  1. 10
  2. 15
  1. import java.util.Scanner;
  2. public class Main {
  3.     public static void main(String[] args) {
  4.     
  5.         Scanner in = new Scanner(System.in);
  6.         
  7.         int num = in.nextInt();
  8.         for (int i = 0; i < num; i++){
  9.             int n = in.nextInt();
  10.             int sum = 0;
  11.             for(int j = 0; j < n; j++){
  12.                 sum += in.nextInt();
  13.             }
  14.             System.out.println(sum);
  15.         }
  16.     }
  17. }

6. 每行第一个为个数后带空格分割整数 (无结束限制,每行有个数限制)

题目描述：

输入数据有多组, 每行表示一组输入数据。
每行的第一个整数为整数的个数n(1 <= n <= 100)。
接下来n个正整数, 即需要求和的每个正整数。

每组数据输出求和的结果

输入示例：

  1. 4 1 2 3 4
  2. 5 1 2 3 4 5

输出示例：

  1. 10
  2. 15
  1. import java.util.Scanner;
  2. public class Main {
  3.     public static void main(String[] args) {
  4.         Scanner in = new Scanner(System.in);
  5.         while (in.hasNextInt()) { 
  6.             int n = in.nextInt();
  7.             int sum = 0;
  8.             while (n > 0) {
  9.                 sum += in.nextInt();
  10.                 n--;
  11.             }
  12.             System.out.println(sum);
  13.         }
  14.     }
  15. }

7. 多组空格分隔的正整数 (无结束限制,每行无个数限制,需要当作字符串处理)

题目描述：

输入数据有多组, 每行表示一组输入数据。
每行不定有n个整数，空格隔开。(1 <= n <= 100)。

每组数据输出求和的结果

输入示例：

  1. 1 2 3
  2. 4 5
  3. 0 0 0 0 0

输出示例：

  1. 6
  2. 9
  3. 0
  1. import java.util.Scanner;
  2. public class Main {
  3.     public static void main(String[] args) {
  4.         Scanner in = new Scanner(System.in);
  5.         while (in.hasNextLine()) { 
  6.             String[] s = in.nextLine().split(" ");
  7.             int sum = 0;
  8.             for (int i = 0; i < s.length; i++) {
  9.                 sum += Integer.parseInt(s[i]);
  10.             }
  11.             System.out.println(sum);
  12.         }
  13.     }
  14. }
  15. // 扩展：第一行组数接空格分开的多个整数 (有行数限制,每行无个数限制,需要当作字符串处理)
  16. // 需要注意的是：读取行数后，要加一行in.nextLine();再读下面的行。
  17. // eg: 每组数据输出求和的结果
  18. // 3
  19. // 1 2 3
  20. // 4 5
  21. // 0 0 0 0 0
  22. import java.util.Scanner;
  23. public class Main {
  24.     public static void main(String[] args) {
  25.         Scanner in = new Scanner(System.in);
  26.         int num = in.nextInt();
  27.         in.nextLine();  // 注意：跳过行数所在行
  28.         for (int i = 0; i < num; i++ ){
  29.             String[] s = in.nextLine().split(" ");
  30.             int sum = 0;
  31.             for(int j = 0; j < s.length; j++){
  32.                 sum += Integer.parseInt(s[j]);  // 将字符转成整数类型
  33.             }
  34.             System.out.println(sum);
  35.         }
  36.     }
  37. }

8. 第一行个数第二行字符串

题目描述：

输入有两行，第一行n
第二行是n个空格隔开的字符串

输出一行排序后的字符串，空格隔开，无结尾空格

输入示例：

  1. 5
  2. c d a bb e

输出示例：

a bb c d e
  1. import java.util.*;
  2. public class Main {
  3.     public static void main(String[] args) {
  4.         Scanner in = new Scanner(System.in);
  5.         int n = in.nextInt();
  6.         in.nextLine();
  7.         while (in.hasNext()) { 
  8.             String[] s = in.nextLine().split(" ");
  9.             Arrays.sort(s);
  10.             for (int i = 0; i < s.length; i++) {
  11.                 System.out.print(s[i] + " ");
  12.             }
  13.             
  14.         }
  15.     }
  16. }

9. 多行空格分开的字符串

题目描述：

多个测试用例，每个测试用例一行。
每行通过空格隔开，有n个字符，n＜100

对于每组测试用例，输出一行排序过的字符串，每个字符串通过空格隔开

输入示例：

  1. a c bb
  2. f dddd
  3. nowcoder

输出示例：

  1. a bb c
  2. dddd f
  3. nowcoder
  1. import java.util.*;
  2. public class Main {
  3.     public static void main(String[] args) {
  4.         Scanner in = new Scanner(System.in);
  5.         while (in.hasNextLine()) { 
  6.             String[] s = in.nextLine().split(" ");
  7.             Arrays.sort(s);
  8.             for (String c:s) {
  9.                 System.out.print(c + " ");
  10.             }
  11.             System.out.println();
  12.         }
  13.     }
  14. }

10. 多行逗号分开的字符串 （逗号和空格的区别就是最后一个字符输完的时候",“还是” " ）

题目描述：

多个测试用例，每个测试用例一行。
每行通过,隔开，有n个字符，n＜100

对于每组用例输出一行排序后的字符串，用','隔开，无结尾空格


输入示例：

  1. a,c,bb
  2. f,dddd
  3. nowcoder

输出示例：

  1. a,bb,c
  2. dddd,f
  3. nowcoder
  1. import java.util.*;
  2. public class Main {
  3.     public static void main(String[] args) {
  4.         Scanner in = new Scanner(System.in);
  5.         while (in.hasNextLine()) { 
  6.             String[] s = in.nextLine().split(",");
  7.             Arrays.sort(s);
  8.             int l = s.length;
  9.             for (int i = 0; i < l - 1; i++) {
  10.                 System.out.print(s[i] + ",");
  11.             }
  12.             System.out.println(s[l-1]);
  13.         }
  14.     }
  15. }

以上就是遇到的各种输入输出的一个总结

三 总结篇

有的读者看完上面的介绍之后，可能还是一脸懵逼，希望提供一个万能输入输出的模版，没关系，既然粉丝有需求，我当然是有求必应：

输入：

  1. test
  2. 3
  3. 1 2 3
  4. 4 5 
  5. 7
  1. import java.util.Arrays;
  2. import java.util.List;
  3. import java.util.Scanner;
  4. import java.util.stream.Collectors;
  5.  
  6. public class Template {
  7.     public static void main(String[] args) {
  8.         Scanner scanner = new Scanner(System.in);
  9.         while (scanner.hasNextLine()) {
  10.             // 获取第一行的输入
  11.             String input = scanner.nextLine();
  12.             // 获取第2行的输入
  13.             int n = Integer.parseInt(scanner.nextLine());
  14.  
  15.             // 获取接下来的N行输入
  16.             for (int i = 0; i < n; ++i) {
  17.                 String[] split = scanner.nextLine().split("\\s+");
  18.                 List<Integer> collect = Arrays.stream(split).map(item -> Integer.parseInt(item)).collect(Collectors.toList());
  19.             }
  20.         }
  21.     }
  22. }

总结一下就是，读取统一采用  scanner.nextLine() 进行获取，

情况1:如果是其他数据类型，则进行数据类型转换；

情况2: 如果一整行包含了多个数据，则整体读取下来之后再使用split进行分隔后二次加工；

情况3: 如果通过读取某一个参数n来读取接下来的n行，则先读取n，并将其转换为整数，之后使用for循环的方式来读取接下来的n行，对于每一行的处理根据情况再选择情况1和情况2的处理方案今进行处理。

学会了这个模版，基本可以应对80% 以上的收入输出的case ，其他的特殊场合如果遇到就去参考分解的方案去应对。
